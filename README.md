  1. 打包大小减少41%
  2. 初次渲染快55%,更新渲染快133%
  3. 内存减少54%

  ## 拥抱TypeScript
  1. Vue3可以更好的支持TypeScript

  ## Composition API (组合API)
  1. setup配置
  2. ref与reactive
  3. watch与watchEffect
  4. provide与inject
